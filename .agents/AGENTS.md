# Project Customization Rules

- **Do Not Use AI-Generated Images:** Do not use AI-generated images on this website. Always search for and link to high-quality online images hosted on stable, public databases (e.g., Kitsu.io, TMDb, or Unsplash).

- **Do Not Push Changes Automatically:** Do not run git push commands to push changes to GitHub/production remote repositories until the user has explicitly verified the changes in local preview and said to push them.
